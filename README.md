README FILE FOR PASTTIMES

Overview
Pastimes is a web-based clothing store application built using PHP and MySQL. It allows users to browse clothing items, add products to a shopping cart, place orders, and manage their account. The system also supports sellers who can submit clothing requests, and an administrator who manages users, products, and seller requests.
 
Features
 
User Features
 
* User registration and login
* View clothing items
* Add items to cart
* Edit cart items and quantities
* Checkout and place orders
* View account details
 
Seller Features
 
* Submit requests to sell clothing items
* Upload product details and images
* Track request status (pending, approved, rejected)
* View seller dashboard
 
Admin Features
 
* Admin login system
* Verify and manage users
* Approve or reject seller requests
* Manage clothing products (add, edit, delete)
 
Project Structure
The project includes PHP files for user, seller, and admin functionality, a database connection file, a stylesheet, and an uploads folder for images.
 
Database
The system uses a MySQL database named ClothingStore. It contains tables for users, admin accounts, clothing items, orders, order history, seller requests, and messages.
 
Installation
The project requires a local server environment such as XAMPP or WAMP. The files should be placed in the server directory and the database imported into phpMyAdmin. After setup, the system can be accessed through a local host URL.
 
Default Admin Access
Admin credentials must be manually inserted into the database before use.
 
Technologies Used
PHP, MySQL, HTML, CSS
 
Purpose
This project was developed for educational purposes to demonstrate full-stack web development concepts including database integration, CRUD operations, and user role management.
