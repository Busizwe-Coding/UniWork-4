README FILE FOR PASTTIMES

Overview
Pastimes is a web-based clothing store application built using PHP and MySQL. It allows users to browse clothing items, add products to a shopping cart, place orders, and manage their account. The system also supports sellers who can submit clothing requests, and an administrator who manages users, products, and seller requests.
 
Technologies Used
PHP, MySQL, HTML, CSS
